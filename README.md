# 🚀 Split - Advanced Group Expense Manager

A high-fidelity, real-time platform for managing group finances, splitting bills, and tracking operational clusters. Built with a focus on premium aesthetics, seamless real-time communication, and robust financial tracking.

## 🌌 Core Features

- **Real-Time Presence**: Instant group activity tracking via Socket.io.
- **Cluster Matrix**: Advanced group management with dynamic settlement algorithms.
- **Interactive Chat**: Real-time signal streams for every operational cluster.
- **Financial Telemetry**: High-fidelity charts and metrics tracking individual and group spending.
- **Quantum UI**: A premium, responsive design system utilizing Indigo/Purple aesthetics and glassmorphism.
- **Identity Registry**: Secure authentication and identity management with JWT.

---

## 🛠 Technology Stack

### **Frontend**
- **React 18** (Vite-powered)
- **Tailwind CSS** (Utility-first styling)
- **Framer Motion** (High-fidelity animations)
- **Recharts** (Data visualization)
- **Lucide React** (Tactical iconography)
- **Socket.io-Client** (Real-time uplink)

### **Backend**
- **Node.js & Express** (Server Core)
- **MySQL** (Relational Data Persistence)
- **MongoDB** (NoSQL Signal Persistence)
- **Socket.io** (Operational WebSocket layer)
- **JWT** (Security & Authorization)

## 🚀 Quick Start

### **1. Clone & Setup**
```bash
git clone https://github.com/Nirmalkumar882000/Frontend_Split_App.git
cd Frontend_Split_App
```

### **2. Backend Initialization**
```bash
cd backend
npm install
# Configure .env with your MySQL and MongoDB credentials
npm start
```

### **3. Frontend Materialization**
```bash
cd frontend
npm install
npm run dev
```

## 🛡️ Operational Governance

The platform utilizes a decoupled architecture where the **Core Engine** (Backend) handles mathematical settlements while the **Command Deck** (Frontend) provides a high-performance visual layer for end-users.

### **API Endpoints**
- `/api/auth`: Identity management
- `/api/groups`: Cluster operations
- `/api/expenses`: Financial telemetry
- `/api/messages`: Real-time signal archive
